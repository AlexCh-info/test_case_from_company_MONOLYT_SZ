### Выходные данные 
В результате оформления заказа, создается папка orders и туда записывается json с уникальным имененем (например, order_20260616_141543.json), чтобы разобраться почему такое название присваивается json файлу, давайте посмотрим из чего оно состоит. Название состоит из двух частей - это дата 20260616 в формате год-месяц-дата, то есть 2026-06-16 и уникального id (141543), который представляет собой время создания заказа, то есть 14часов-15минут-43секунд. В дальнейшем лучше его заменить на автогенератор id без привязки ко времени и другим показателям, чтобы id двух заказов не мог повториться.
```
# Пример json
{
    "material": "Газобетон ЛСР D400",
    "category": "Газобетон",
    "region": "МСК",
    "price": 550,
    "created_at": "2026-06-16T14:15:43.658726"
}
```
